jetleaf_devtool

Development tooling for JetLeaf — source generation helpers, runtime inspection utilities, and CLI automation.

Package: jetleaf_devtool (includes a small CLI executable jl)

Key features

  • Source generation helpers (build + source_gen integration).
  • Runtime inspection utilities and VM service integration.
  • Developer CLI for common tasks (jl dev, jl generate, jl build).

Documentation: https://jetleaf.hapnium.com/docs/devtool Repository: https://github.com/jetleaf/jetleaf_devtool
